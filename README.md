# Тестовый проект: Каталог товаров

Демо-приложение на React 19 с TypeScript, Ant Design, Emotion и TanStack Query. Реализована авторизация, каталог товаров с пагинацией, сортировкой и поиском, а также модальное окно добавления товара.

🔗 **Демо**: https://productstabletestapp.vercel.app
Рекомендуется использовать VPN для стабильной работы Dummy API

---

## 🚀 Технологии и инструменты

- **React 19** — современная библиотека для UI
- **TypeScript** — типизация и улучшенная разработка
- **Ant Design (v5)** — библиотека компонентов
- **Emotion** — CSS-in-JS стилизация
- **TanStack Query** — управление серверным состоянием, кэширование
- **React Router v6** — навигация и защита маршрутов
- **Webpack** — сборка проекта
- **ESLint + Prettier** — качество и форматирование кода
- **DummyJSON** — тестовое API для товаров и авторизации

---

## 📁 Архитектура (Feature-Sliced Design)

Проект организован по методологии **FSD**, что обеспечивает масштабируемость и лёгкость поддержки.

---

## ✨ Реализованные функции

### 🔐 Авторизация
- Форма входа с валидацией
- Чекбокс «Запомнить данные» — токен сохраняется в `localStorage` или `sessionStorage`
- Кнопка выхода (сбрасывает токен и возвращает на логин)

### 📦 Страница товаров
- **Таблица** с серверной пагинацией (5 элементов на странице)
- **Сортировка** по колонкам: наименование, вендор, оценка, цена (через API)
- **Поиск** с выпадающим списком — показывает название и цену (дебаунс 300 мс)
- **Рейтинг**: если значение ниже 3.0, число подсвечивается красным (например, `2.4/5`)
- **Модальное окно** добавления товара (имитация, без реального API)
- **Глобальные CSS-переменные** для цветов (легко менять тему)
- **Кастомные шрифты** Inter (основной), Cairo, Open Sans, Roboto Mono

### 🎨 Дизайн
- Вёрстка по макету Figma
- Адаптивные компоненты Ant Design с кастомизацией через `ConfigProvider` и Emotion
